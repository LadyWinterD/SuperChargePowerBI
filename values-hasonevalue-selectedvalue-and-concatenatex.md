# VALUES\(\),HASONEVALUE\(\),SELECTEDVALUE\(\),and CONCATENATEX\(\)

### VALUES\(\)

```text
Total Months in Calendar = COUNTROWS(
VALUES('Calendar'[MonthName])
)
```

```text
Month Name(Values) = IF(
HASONEVALUE('Calendar'[MonthName]),VALUES('Calendar'[MonthName])
)
```
